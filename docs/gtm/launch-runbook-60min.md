# Launch Runbook (60 Minutes)

## Objective
Execute a coordinated launch immediately after benchmark sign-off with one metric-injection pass.

## Timeline
1. T+0: inject metrics from finalized snapshot into all drafts.
2. T+10: publish Medium post.
3. T+20: publish LinkedIn post.
4. T+25: publish X launch thread.
5. T+35: open first integration PR wave.
6. T+50: monitor replies, issues, and PR feedback.

## Operator Commands
1. Confirm CTA pair appears in each publish-facing draft.
2. Confirm placeholders have no remaining unresolved tokens.
3. Confirm all numeric values match snapshot exactly.

## Abort Conditions
1. Snapshot missing required schema keys.
2. Conflicting values detected across drafts.
3. Any draft includes unsupported numeric claim.
